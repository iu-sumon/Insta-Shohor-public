JavaScript is an interpreted language that means javascript code is executed in line by line manner.We can understand how a typical JavaScript engine works .JavaScript engine the source code goes through several steps.Let us understand each of these steps. Step-1:Parser(our code is first received by the "parser".The parser's job is to check the JavaScript code for syntactic errors in line by line manner ) Step-2:AST(Once the parser checks all JavaScript codes and gets satisfied that there are no mistakes/errors in the code, it creates the data structure called AST (it stands for Abstract Syntax Tree).) Step-3:Converted (0,1) The JavaScript engine converts the JavaScript code into the machine code Step-4:Machine Code(The converted code is sent to the system for execution, and finally, that byte code run by the system/engine)

JavaScript is single-threaded .JavaScript is non-blocking .JavaScript is a scripting language for the web.It is an interpreted language, which means it does not need a compiler to translate its code.JavaScript code runs directly in a web browser.The event loop is a process that waits for the Call Stack to be clear before pushing callbacks from the Task Queue to the Call Stack.Once the Stack is clear,the event loop triggers and checks the Task Queue for available callbacks.If there are any, it pushes it to the Call Stack, waits for the Call Stack to be clear again, and repeats the same process.
